# MAKEFILE

The `MAKEFILE` is used to save and automate certain commands and tasks.

## Usage

```bash
make <command>
```

## Commands

| Command  | Description           |
| -------- | --------------------- |
| `start`  | Start the application |
| `build`  | Build the application |
| `test`   | Run the tests         |
| `format` | Format the code       |
| `clean`  | Clean the project     |
